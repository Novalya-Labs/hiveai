# What is HiveAI?

HiveAI is a **TypeScript framework for building autonomous AI agents** that work together in teams. Think of it as an orchestration layer that lets multiple AI agents collaborate, share data, and execute complex workflows.

## Key Features

  <Card title="Agent Orchestration" icon="sitemap">

## Why HiveAI?

### Problem

Building multi-agent systems is complex:
- Managing agent dependencies and execution order
- Sharing data between agents
- Handling multiple LLM providers
- Configuring prompts and tools
- Debugging agent interactions

### Solution

HiveAI provides:
```yaml
# Simple YAML configuration
name: research-agent
llm:
  provider: mistral
  temperature: 0.7
depends_on: data-collector
tools:
  - firecrawl
  - file-reader
goals:
  - Extract insights from collected data
```

## Quick Example

```yaml
# teams/my-pipeline/researcher.yml
name: researcher
llm: mistral
tools:
  - firecrawl
goals:
  - Scrape and analyze website content
tasks:
  - Extract key information
  - Summarize findings
```

# Run the pipeline
