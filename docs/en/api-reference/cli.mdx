# CLI Reference

Complete reference for the HiveAI CLI.

## Commands

### `hiveai run <team>`

Run all agents in a team.

```bash
npx hiveai run my-team
```

### `hiveai help`

Show help message.

```bash
npx hiveai help
```

### `hiveai agent add <name>`

Create a new agent (coming soon).

### `hiveai team add <name>`

Create a new team directory (coming soon).

## Output

The CLI provides:
- Real-time execution logs
- Progress indicators
- Error messages
- Performance metrics
- Pipeline summary

## Exit Codes

- `0`: Success
- `1`: Failure (agent error, validation error, etc.)
