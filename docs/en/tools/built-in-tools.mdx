# Built-in Tools

HiveAI includes several built-in tools ready to use.

## Available Tools

<CardGroup cols={2}>
  <Card title="file-reader" icon="file">
    Read JSON, CSV, TXT, MD files
  </Card>
  <Card title="firecrawl" icon="globe">
    Web scraping with Firecrawl API
  </Card>
  <Card title="serializer" icon="table">
    Convert data to CSV
  </Card>
</CardGroup>

## Usage

```yaml
tools:
  - file-reader
  - firecrawl
  - serializer
```

The LLM automatically gets descriptions and knows when to use each tool.
